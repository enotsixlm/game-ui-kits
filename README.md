# 🎮 Game UI Kits · 游戏 UI 组件库

六套不同风格的游戏界面组件库，包含按钮、面板、血条/进度条、弹窗、道具栏等常用游戏 UI 组件。

**零依赖、零构建**：每套都是一个自包含的单文件 HTML + CSS，唯一的外部资源是 Google Fonts 字体。浏览器打开即可预览，样式可直接复制进你的项目。

## 📦 使用方式

1. 打开对应风格的页面，找到想要的组件；
2. 每套组件库顶部定义了一组 design tokens（CSS 变量），先把 `:root` 里的变量复制过去；
3. 再复制组件对应的 HTML 结构和 CSS 类即可。
